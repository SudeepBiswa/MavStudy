**IDE: Visual Studio Code (VS Code) **  

**Extension: Live Server**  
&ensp;- After installing Live Server, you will see "Port: {num}" or "Go Live" at the bottom right of VS Code. Clicking on this will start/stop the live server.  
&ensp;- To open the Live server on a webpage, right-click the HTML file and click "Open with Live Server."

**The form.js file has code establishing a connection to a Firebase server owned by Sudeep. PLEASE DO NOT ABUSE THIS. ONLY USE IT FOR THIS PROJECT**.

The Live Server extension is needed to communicate with Firebase.

Installing Live Server Extension:  
&ensp;- Open VS Code  
&ensp;- Look at the left side towards the top  
&ensp;- Click on the option with the 3 squares and one angled square  
&ensp;- This is the marketplace, search for "Live Server"
